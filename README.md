Data Engineering Barista ☕

Applying a data engineering mindset to coffee brewing.

## Project Overview

Data Engineering Barista is a coffee experimentation website that combines measurement, observation, and analysis to improve brewing consistency.

Each brew is treated like a small data pipeline. Inputs are collected, transformed through the brewing process, measured, documented, and analyzed to guide future experiments.

## Features

- Brew Log
- Coffee Photos
- Experiment Notes
- Brewing Workflow Visualization
- Static Website hosted on GitHub Pages

## Technologies

- HTML
- CSS
- GitHub Pages

## Current Experiment

Brew #001 tested an E&B Lab by IMS Competition Moka Filter together with an AeroPress paper filter in a Bialetti 3-Cup Moka Pot.

Initial observations showed a cleaner and smoother cup with reduced sediment.
